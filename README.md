## 📌 Overview

**VitalMind** is an AI-powered system designed to detect stress and burnout risk in real time using digital biomarkers such as Heart Rate Variability (HRV), sleep patterns, heart rate, and physical activity.

It leverages personalized baselines and intelligent scoring to provide actionable insights for better mental health monitoring.

---

## 🚀 Features

* 🔹 Real-time burnout risk scoring
* 🔹 7-day risk trend analysis
* 🔹 Signal-wise breakdown (HRV, Sleep, HR, Activity)
* 🔹 Interactive wearable simulator
* 🔹 Personalized, baseline-based predictions

---

## 🧪 How It Works

1. Collect biomarker data (HRV, sleep, heart rate, steps)
2. Compare with personal baseline
3. Apply weighted scoring model
4. Generate stress risk score (0–100)
5. Provide insights and recommendations
